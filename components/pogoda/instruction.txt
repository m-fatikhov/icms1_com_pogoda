Как узнать ID своего города?

1-й способ:
1. Скачиваем файл http://bulk.openweathermap.org/sample/city.list.json.gz .
2. Разархивируйте скачанный файл.
3. Внутри архива найдете файл city.list.json. Откройте его в любом текстовом редакторе.
4. Поиском найдите строку со своим городом. В поле поиска нужно вбивать английское название города.
   Например, для Москвы будет следующая строка:
   {"_id":524901,"name":"Moscow","country":"RU","coord":{"lon":37.615555,"lat":55.75222}}
   Значит, ID Москвы равен 524901.


2-й способ:
1.  Перейдите на сайт http://openweathermap.org .
2.  Прямо по середине страницы найдите поле поиска (где написано: Your city name и кнопочка Search). Введите название Вашего
    города, можно на русском.
3.  В результатах поиска найдите Ваш город и перейдите по ссылке.
4.  Откроется страница Вашего города. При этом в адресной строке браузера будет что-то подобное:
    http://openweathermap.org/city/524901  - для Москвы.
    Цифры за последним слэшем(524901) и есть ID нужного города.
